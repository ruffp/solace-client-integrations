# Documentation of the provisioning section

## Scripts

### 1. Python scripts

Python scripts are used for calling Solace SEMPv2 API:

* create-queue.py
* delete-queue.py
* patch-queue.py
* subscribe-queue.py

### 2. Bash scripts

Bash scripts are used for calling python script for queue creation:

* install.sh 
* uninstall.sh
